It is a search engine (Crawler,indexer) project. To run this project you should only run the Main class and enter the number of threads you want when it is required.
In order to open index.html you should download nodejs and open the Apache
You should make a folder in C:/Program  Files/nodejs and then put in it index.html and app.js, then open the cmd in administrator mode and go to the directory of this folder then write node app.js, then you can open index.html
You should also import the libraries uploaded here
